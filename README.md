# OpenAPI Python CLI

- export `OPENAI_API_KEY` in env
- EXEC: `python run.py`
  - input an IMAGE description 
- Output is saved in `images`

```bash
$ export OPENAI_API_KEY ='sk-*********************'
$ set    OPENAI_API_KEY ='sk-*********************'
$ $env:OPENAI_API_KEY   ='sk-*********************'
```
